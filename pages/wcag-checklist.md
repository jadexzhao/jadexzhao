# wcag accessibility checklist

*based on wcag 2.1 aa/aaa standards. used across serve it and consulting work.*

---

**perceivable**
- [ ] all images have descriptive alt text
- [ ] color is not the only means of conveying information
- [ ] text contrast ratio meets 4.5:1 (aa) or 7:1 (aaa)
- [ ] captions provided for video content
- [ ] content readable at 200% zoom without horizontal scroll

**operable**
- [ ] all functionality accessible by keyboard
- [ ] no keyboard traps
- [ ] skip navigation link provided
- [ ] focus indicators visible
- [ ] no content flashes more than 3 times per second

**understandable**
- [ ] language of page declared in html
- [ ] labels provided for all form inputs
- [ ] error messages are descriptive and helpful
- [ ] consistent navigation across pages

**robust**
- [ ] valid, semantic html
- [ ] aria labels used correctly and sparingly
- [ ] tested with screen reader (nvda, voiceover)
- [ ] tested with keyboard-only navigation

---

*this is a starting point, not a ceiling.*
